Личные проекты

Инструментарий для создания telegram-ботов формата одного сообщения

Разработал инструментарий для создания Telegram-ботов с одноэкранной моделью взаимодействия. Спроектировал модульную архитектуру, позволяющую быстро собирать сценарные и игровые решения на основе переиспользуемых компонентов. Реализовал управление пользовательскими сценариями через state machine (map-based routing), систему работы с сообщениями, клавиатурами и медиаконтентом, а также поддержку многоязычности. Добавил встроенные механики монетизации, включая платежи, токены и реферальную систему. Обеспечил конфигурацию приложения через переменные окружения и структуру проекта, ориентированную на масштабируемость и быстрое создание новых ботов.
